Observations

  • Every internal VLAN has a “Temporary allow all - migration” pass rule on OPNsense.
  • MikroTik has no L3 firewall rules; all inter-VLAN filtering is delegated to OPNsense.
  • VLAN 45 (Management) has four untagged access ports on MikroTik: ether1, ether5, ether9, ether12.
  • MikroTik bridge retains legacy 10.0.1.34/16 address.
  • VLAN 71 (ISPRouter / 10.67.81.0/24) is present but not in original Blackwall design.
  • 31 public hostnames are exposed through Traefik on Infrastructure host (grafana/infisical/authentik) or Services host (all others).
  • 30 compose-files repos define deployed stacks; ai-stack, synapse, arr-stack, immich, authentik are multi-service.
  • Live network-flows job captured 11 hosts with 230 listening sockets (116 publicly bound).
  • Port 18080 is the most common public bind, likely Traefik metrics or a common app port.
  • Port 22 (SSH) is open on most hosts; port 25 (postfix/master) appears on several hosts bound to 127.0.0.1 only.
  • Port 9100 (node_exporter) is present across infrastructure/hosts.
  • Docker proxy processes bind the vast majority of public ports; host networking is minimal.

Recommended Next Steps
